# GASTRONOM v2
### It's improved version of [GASTRONOM v1](https://github.com/kuchersd/shop1c)

## Details:

- WPF Application
- Target version: .netcore 3.1
- UI: MaterialDesign XAML [GitRepo](https://github.com/MaterialDesignInXAML/MaterialDesignInXamlToolkit)
- Database: Local MS SQL Database
- Data access technology: Entity Framework Core
